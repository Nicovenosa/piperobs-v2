Procesos y Threads

Un proceso es un programa en ejecución. Cada proceso tiene su propio espacio de memoria, registros y estado.

Un thread (hilo) es la unidad mínima de ejecución dentro de un proceso. Varios threads pueden compartir el mismo espacio de memoria.

Diferencias clave

| Proceso | Thread |

| Memoria separada | Memoria compartida |

| Más pesado | Más liviano |
